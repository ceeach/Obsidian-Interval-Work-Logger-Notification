# **Obsidian Interval Work Logger**

A custom Obsidian plugin that prompts you at regular intervals to log your work and perfectly syncs with the Day Planner timeline.

---

## Features

- **Interval Prompts**: Get notified at your configured intervals (e.g., every 5, 30, or 60 minutes) to log what you've been working on.
- **Day Planner Integration**: Automatically formats your logs as Markdown task blocks (e.g.,  13:00 - 14:00 Finished report) under your # Day planner heading so they appear visually in your timeline.
- **Remind Me Later**: Busy? Use the "Remind Me Later" button to delay the prompt.
- **Single-Instance Modal**: Built-in safeguards ensure that double-clicking notifications won't spam your screen with multiple input windows.

## Installation

1. Copy the plugin folder into your vault's .obsidian/plugins/ directory.
2. Open Obsidian and go to Settings > Community Plugins
3. Disable "Safe Mode" if it is enabled.
4. Find Obsidian Interval Work Logger in the list and toggle it on.

## How to Use
Once enabled, the plugin will run in the background.
1. A notification will appear when your interval is up.
2. Click the notification to open the ObsidianIntervalWorkLoggerNotification check-in window.
3. Type in what you accomplished and hit Log Entry.
4. Check your Daily Note—the time block is automatically logged and ready for the Day Planner!
